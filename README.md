# Engineering Progress Log

A React blog for documenting engineering progress, project notes, and learning milestones while building toward junior developer work.

## Tech Stack

- React
- Vite
- JavaScript / JSX
- CSS
- Lucide React
- localStorage

## Features

- Responsive engineering blog layout
- AI/computer-inspired visual theme
- Reusable post preview component
- New log entry form
- Persistent entries with browser localStorage
- Reset button for returning to starter data

## Learning Goals

This project is being built as both a learning tool and portfolio piece. It focuses on job-relevant React fundamentals:

- Components
- Props
- State
- Events
- Controlled forms
- Lists with `.map()`
- Effects with `useEffect`
- Browser persistence with localStorage

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Build

```bash
npm run build
```

## Portfolio Notes

This app is intended to grow into a public engineering journal that can support junior developer applications and Upwork proposals by showing consistent progress, project thinking, and practical frontend skills.
